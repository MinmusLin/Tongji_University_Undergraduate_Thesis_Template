# Tongji University Undergraduate Thesis Template

## 仓库名称

Tongji_University_Undergraduate_Thesis_Template

## 仓库简介

Tongji university undergraduate thesis template (STEM).

同济大学本科毕业设计（论文）模板（理工类）。

## 仓库组成

* `/figures`
图片文件

* `/fonts`
字体文件

* `/sections`
章节文件

* `/style`
样式文件

* `main.pdf`
示例 PDF 文档文件

* `main.tex`
主文档文件

